# sing-box sidecar

This directory contains the real sing-box sidecar for the current Windows
MSVC target:

```text
sing-box-x86_64-pc-windows-msvc.exe
```

Current version:

```text
sing-box 1.13.12 windows/amd64
```

For other target platforms, add the matching target-triple binary name required
by Tauri v2, for example `sing-box-aarch64-apple-darwin`.
