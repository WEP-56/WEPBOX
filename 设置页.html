开机自动运行 写入系统登录自启注册表，随电脑启动静默拉起客户端。
应用启动后自动拉起内核 打开软件时即自动启动 sing-box 核心，无需手动点击首页开关。
启动时隐藏主窗口 自启或打开时不弹出主界面，直接无感进入系统托盘后台。
关闭窗口时隐藏到托盘 点击主窗口关闭按钮时彻底避免误关软件，退出请在右下角托盘菜单操作。
异常退出时通知 若内核发生崩溃或意外闪退中断，在系统右下角弹出预警提示。
本地 Mixed 端口 HTTP / SOCKS 混合代理入口，非 TUN 模式下将被用于配置系统代理。
Clash API 端口 节点页、外部控制、延迟测速及运行探活依赖的核心外部控制端口。
Clash API Secret 用于本地 API 的安全鉴权密钥。留空时后台会自动生成随机 UUID。
允许局域网连接 开启后监听地址从 127.0.0.1 变为 0.0.0.0，允许同局域网的其他设备接入。
规则模式 Fallback 策略 当规则未命中任何匹配时，决定默认兜底走直连还是走代理。
自定义 DNS 服务器 每行一个地址。第一行优先作为直连国家域名解析，第二行作为远程。
Fake-IP 虚拟网段 (IPv4) 用于 fake-dns 返回的虚拟内部局域网分配池。
TUN 虚拟网卡设备名称 写入系统的虚拟网络适配器设备名，可在网络连线中可见。
Strict Route (严格路由) 更严格地强制限制系统流量全数由 TUN 接口接管，防止任何侧漏。
第三方订阅转换 URL 当本地无法解析标准订阅格式时，通过远程后端安全转码为 sing-box 兼容的 JSON。
订阅自动更新间隔 后台定时无感下发拉取云端节点，失败将写入事件日志。
自动选择最优节点 测速计划完成后，后台将自动为策略分组切向延迟最低的极速可用节点。
